# Homebrew Deprecated

- php 5.6
- php 7.0

Homebrew tap for deprecated formulae.

## Installation

```bash
brew tap kunte/homebrew-deprecated
```

## Usage

```bash
brew install <formula>
```

See [Formula](Formula) directory for available formulae.
